# MonoDB Admin

This is where development for MonoDB Admin takes place. MonoDB Admin is a Tauri-based desktop application that allows you to manage your MonoDB databases.

It is currently extremely limited in what it can do, but the goal is to eventually have it support all of the features of MonoDB.

## Building

```bash
# Make sure you have yarn installed
yarn

# To build the application
yarn tauri build

# To run the application in development mode
yarn tauri dev
```
